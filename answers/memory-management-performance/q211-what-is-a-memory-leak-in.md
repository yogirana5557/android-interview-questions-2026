# Question 211: What is a Memory Leak in Android?

**Category:** Memory Management & Performance  
**Difficulty:** Junior

---

### Answer

A memory leak occurs when an object that is no longer needed by the application remains referenced by another object that continues to exist. This prevents the Garbage Collector (GC) from reclaiming its memory, leading to increased memory usage and eventual OutOfMemoryError (OOME).
