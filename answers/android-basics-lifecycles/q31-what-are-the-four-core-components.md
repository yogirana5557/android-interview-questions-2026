# Question 31: What are the four core components of an Android application?

**Category:** Android Basics & Lifecycles  
**Difficulty:** Junior

---

### Answer

Android applications are built using four key components, which must be declared in the `AndroidManifest.xml`:* **Activities**: The entry points for user interaction, representing a single screen with a user interface.
* **Services**: Components that run in the background to perform long-running operations without a user interface (e.g., music playback).
* **Broadcast Receivers**: Components that enable the system or other apps to deliver events/announcements to the app (e.g., low battery notification).
* **Content Providers**: Manage access to a structured set of data, sharing data between different applications securely.
