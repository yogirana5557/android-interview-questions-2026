# Question 241: What is the difference between Local Unit Tests and Instrumented Tests?

**Category:** Testing & Automation  
**Difficulty:** Junior

---

### Answer

* **Local Unit Tests**: Run on the local development JVM (computer). They do not require a physical device or emulator and compile quickly. Mock frameworks (like Mockito/MockK) are used to mock Android dependencies.
* **Instrumented Tests**: Run on a physical Android device or emulator. They compile into a separate test APK and test integration with actual Android SDK components and UI layouts.
