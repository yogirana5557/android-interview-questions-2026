# Question 111: What is the difference between Imperative UI and Declarative UI?

**Category:** State Management & Jetpack Compose  
**Difficulty:** Junior

---

### Answer

* **Imperative UI** (Classic Views): You manually construct and modify the UI widget tree in response to data changes using methods like `setText()` or `setVisibility()`. You manage the view state manually.
* **Declarative UI** (Jetpack Compose): You describe what the UI should look like for a given state. When the state changes, the framework automatically recreates or updates the UI to match the new state.
