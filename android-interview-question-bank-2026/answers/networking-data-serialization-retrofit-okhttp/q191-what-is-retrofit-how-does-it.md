# Question 191: What is Retrofit? How does it differ from standard HttpUrlConnection?

**Category:** Networking & Data Serialization (Retrofit, OkHttp)  
**Difficulty:** Junior

---

### Answer

`Retrofit` is a type-safe HTTP client for Android and Java. Unlike `HttpUrlConnection` (which requires manual thread management, stream reading, and JSON parsing), Retrofit abstracts the network calls into simple interfaces with annotations, converting HTTP responses into typed objects automatically using converter factories.
