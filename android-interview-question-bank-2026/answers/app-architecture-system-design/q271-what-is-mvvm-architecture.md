# Question 271: What is MVVM architecture?

**Category:** App Architecture & System Design  
**Difficulty:** Junior

---

### Answer

MVVM (Model-View-ViewModel) separates UI code from business logic:* **View**: Displays the UI and routes user events to the ViewModel.
* **ViewModel**: Holds UI state and business logic, communicating state changes to the View via observable streams (Flow/LiveData).
* **Model**: Handles data access (network, local database).
